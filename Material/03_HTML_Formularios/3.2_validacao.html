📚 Conceitos de Validação HTML5

✅ Vantagens da Validação HTML5:

  • Instantânea: Validação em tempo real
  • Nativa: Funciona sem JavaScript
  • Acessível: Compatível com tecnologias assistivas
  • Consistente: Comportamento padrão entre navegadores
  • Personalizável: Mensagens e estilos customizáveis

⚠️ Limitações:

  • Validação apenas no cliente (sempre validar no servidor)
  • Suporte limitado em navegadores muito antigos
  • Mensagens padrão podem não estar em português
  • Regras de validação limitadas comparado com JavaScript

🎯 Atributos de Validação

Required - Campo Obrigatório

Este campo é obrigatório
<input type="text" required placeholder="Campo obrigatório">

Minlength/Maxlength - Comprimento

0/20 caracteres
Deve ter entre 3 e 20 caracteres
<input type="text" minlength="3" maxlength="20">

Min/Max - Valores Numéricos

Idade deve estar entre 18 e 65 anos
<input type="number" min="18" max="65">

Step - Incrementos

Deve ser múltiplo de 0.50
<input type="number" step="0.50" min="0">

🔍 Validação com Padrões (Pattern)

Código Postal Português

Formato: 1234-567
Formato inválido. Use: 1234-567
<input type="text" pattern="[0-9]{4}-[0-9]{3}" title="Formato: 1234-567">

Telemóvel Português

Formato: 9XXXXXXXX
Número de telemóvel inválido
<input type="tel" pattern="(9[1236])[0-9]{7}">

Nome Utilizador

3-16 caracteres: letras, números, underscore
Apenas letras, números e _ (3-16 caracteres)
<input type="text" pattern="[a-zA-Z0-9_]{3,16}">

Password Forte

Mín. 8 chars: maiúscula, minúscula, número, símbolo
Password deve ser mais forte
pattern="(?=.*[a-z])(?=.*[A-Z])(?=.*\d)(?=.*[@$!%*?&])[A-Za-z\d@$!%*?&]{8,}"

⚡ Validação com JavaScript

✅ Vantagens da Validação JavaScript:

  • Mensagens de erro personalizadas
  • Validação em tempo real
  • Regras de negócio complexas
  • Melhor experiência do utilizador
  • Integração com APIs externas
// Exemplo de validação personalizada const emailInput = document.getElementById('email'); emailInput.addEventListener('input', function() { const email = this.value; const isValid = /^[^\s@]+@[^\s@]+\.[^\s@]+$/.test(email); if (isValid) { this.setCustomValidity(''); showSuccess('Email válido!'); } else { this.setCustomValidity('Email inválido'); showError('Formato de email incorreto'); } }); // Validação de passwords coincidentes function validatePasswordMatch() { const password = document.getElementById('password').value; const confirm = document.getElementById('confirm-password').value; if (password !== confirm) { document.getElementById('confirm-password').setCustomValidity('Passwords não coincidem'); return false; } else { document.getElementById('confirm-password').setCustomValidity(''); return true; } }
