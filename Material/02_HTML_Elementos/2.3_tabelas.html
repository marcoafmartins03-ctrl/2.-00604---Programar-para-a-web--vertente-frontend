Acessibilidade:
  • <caption> - Descreve o propósito da tabela
  • scope="col/row" - Define se o cabeçalho é de coluna ou linha
  • headers - Associa células aos seus cabeçalhos
  • id - Identificadores únicos para referência

💡 Dicas e Boas Práticas

✅ Boas Práticas:

  • Use tabelas apenas para dados tabulares, não para layout
  • Sempre inclua cabeçalhos (<th>) para identificar colunas
  • Use <caption> para descrever o propósito da tabela
  • Mantenha as tabelas simples e legíveis
  • Use zebra striping (linhas alternadas) para melhor leitura

❌ Evite:

  • Usar tabelas para layout de página
  • Tabelas sem cabeçalhos
  • Mesclar células desnecessariamente
  • Tabelas muito largas em dispositivos móveis
  • Texto muito pequeno em células

CSS Básico para Tabelas:

/* Estilo básico */ table { border-collapse: collapse; /* Remove espaços entre bordas */ width: 100%; } th, td { border: 1px solid #ddd; padding: 12px; text-align: left; } th { background-color: #f2f2f2; font-weight: bold; } /* Zebra striping */ tr:nth-child(even) { background-color: #f9f9f9; } /* Hover effect */ tr:hover { background-color: #f5f5f5; }
