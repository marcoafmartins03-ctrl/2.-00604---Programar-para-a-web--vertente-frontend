Lição 9.2: Condicionais

Instruções

  1. Abre a Consola do Browser (F12 → Console)
  2. Vê os resultados do JavaScript
  3. O código está em 9.2_condicionais.js

Conceitos desta lição:

  • if — executar código se condição for verdadeira
  • else — alternativa quando condição é falsa
  • else if — múltiplas condições
  • Operadores: ===, !==, >, <
