Bem-vindo ao Tailwind CSS

Tailwind é diferente do Bootstrap. Em vez de componentes prontos, usas classes utilitárias para construir qualquer design. Cada classe faz uma coisa específica.

💡 Comparação com Bootstrap

Bootstrap: class="btn btn-danger"

Tailwind: class="bg-red-500 text-white px-4 py-2 rounded"

Tailwind dá mais controlo, Bootstrap é mais rápido para protótipos.
