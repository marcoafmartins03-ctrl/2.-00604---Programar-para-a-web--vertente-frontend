# Bibliotecas de componentes para Tailwind (com exemplos)

## Objetivo
Este documento lista bibliotecas/kit de componentes para Tailwind CSS e inclui exemplos curtos para ajudar os formandos a perceber o estilo e as dependencias.

## Como escolher rapidamente
- Se queres so HTML + classes e zero JS: prefere kits "copy/paste" (HyperUI, Meraki, Mamba, Flowrift).
- Se precisas de interacao (dropdown, modal, tabs): usa libs com JS (Flowbite, Preline, Tailwind Elements).
- Se estas a trabalhar com React: considera Headless UI ou shadcn/ui.
- Se queres componentes oficiais e suporte: Tailwind UI (pago).

## Bibliotecas e kits recomendados
- DaisyUI: plugin Tailwind com classes semanticas (btn, card, navbar). Gratis + temas.
- Flowbite: componentes com JS opcional; bom para dropdowns/modals. Gratis + Pro.
- Preline: componentes com JS, focado em dashboards.
- Tailwind UI: oficial e pago, qualidade alta.
- Headless UI: componentes acessiveis sem estilo (React/Vue), estilo com Tailwind.
- shadcn/ui: componentes React + Tailwind + Radix (copy/paste).
- HyperUI: componentes free, copy/paste.
- Meraki UI: componentes free, copy/paste.
- Mamba UI: componentes free, copy/paste.
- Flowrift: componentes marketing free.
- Tailwind Elements (tw-elements): componentes com JS, estilo Bootstrap-like.
- Windmill UI: kit free para dashboards.

## Exemplos rapidos (HTML)
### DaisyUI - Card + botao
```html
<div class="card w-80 bg-base-100 shadow-xl">
  <figure>
    <img src="img/produto.jpg" alt="Produto exemplo">
  </figure>
  <div class="card-body">
    <h3 class="card-title">Produto simples</h3>
    <p>Descricao curta para o card.</p>
    <div class="card-actions justify-end">
      <button class="btn btn-primary">Ver mais</button>
    </div>
  </div>
</div>
```
Nota: requer plugin DaisyUI na configuracao do Tailwind.

### Flowbite - Dropdown simples
```html
<button data-dropdown-toggle="dropdownMenu" class="text-white bg-blue-700 px-4 py-2 rounded">
  Menu
</button>

<div id="dropdownMenu" class="hidden z-10 bg-white divide-y divide-gray-100 rounded-lg shadow w-44">
  <ul class="py-2 text-sm text-gray-700">
    <li><a href="#" class="block px-4 py-2 hover:bg-gray-100">Perfil</a></li>
    <li><a href="#" class="block px-4 py-2 hover:bg-gray-100">Definicoes</a></li>
  </ul>
</div>
```
Nota: requer o script JS do Flowbite para ativar o dropdown.

### Preline - Accordion simples
```html
<div class="hs-accordion-group">
  <div class="hs-accordion">
    <button class="hs-accordion-toggle w-full text-left px-4 py-2 border rounded">
      O que e Tailwind?
    </button>
    <div class="hs-accordion-content hidden px-4 py-2">
      Tailwind e um framework utility-first para CSS.
    </div>
  </div>
</div>
```
Nota: requer o script JS do Preline para abrir/fechar o accordion.

## Exemplo (React)
### Headless UI - Menu basico
```jsx
import { Menu } from "@headlessui/react";

<Menu as="div" className="relative inline-block">
  <Menu.Button className="bg-gray-900 text-white px-4 py-2 rounded">Acoes</Menu.Button>
  <Menu.Items className="absolute mt-2 w-40 bg-white shadow rounded">
    <Menu.Item>
      {({ active }) => (
        <a className={`block px-3 py-2 ${active ? "bg-gray-100" : ""}`} href="#">
          Editar
        </a>
      )}
    </Menu.Item>
  </Menu.Items>
</Menu>
```
Nota: Headless UI e para React/Vue, nao e HTML puro.

## Sugestao para aula

- Explicar que o objetivo e acelerar, mas a base continua a ser HTML/CSS bem estruturado.
