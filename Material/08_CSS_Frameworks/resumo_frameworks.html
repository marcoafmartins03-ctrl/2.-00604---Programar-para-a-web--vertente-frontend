CSS Frameworks — Resumo Visual

Bootstrap vs Tailwind

Aspeto Bootstrap Tailwind
Abordagem Componentes prontos Classes utilitárias
O que adiciona Classes CSS + JS opcional; não cria tags HTML novas Classes CSS utilitárias; sem biblioteca JS oficial de componentes
Botão btn btn-primary bg-blue-500 text-white px-4 py-2 rounded
Grid row + col-md-6 grid grid-cols-2 ou flex
Personalização Limitada (override CSS) Total (classes combinadas)
Ideal para Protótipos rápidos, dashboards Designs únicos, controlo total

Bootstrap — Classes Essenciais

Container e Grid

<div class="container"> <div class="row"> <div class="col-md-6">Metade</div> <div class="col-md-6">Metade</div> </div> </div>

Cores

text-primary text-success text-danger text-warning

bg-primary bg-success bg-danger bg-warning

Spacing (p-, m-)

p-1 a p-5 | m-1 a m-5 | mt-3 mb-3 mx-auto

Componentes

btn btn-primary | card card-body | alert alert-info | badge bg-success

Tailwind — Classes Essenciais

Layout

<div class="max-w-4xl mx-auto px-4">Container</div> <div class="flex gap-4">Flexbox</div> <div class="grid grid-cols-3 gap-4">Grid</div>

Cores (cor-intensidade)

text-blue-500 text-green-500 text-red-500 text-gray-700

bg-blue-500 bg-green-500 bg-red-500 bg-gray-100

Spacing

p-4 padding | px-6 horizontal | py-2 vertical | m-4 margin | mb-6 bottom

Botão (exemplo)

<button class="bg-blue-500 text-white px-4 py-2 rounded hover:bg-blue-600"> Botão </button>

Card (exemplo)

<div class="bg-white rounded-lg shadow p-6"> <h3 class="text-xl font-bold">Título</h3> <p class="text-gray-600">Conteúdo...</p> </div>

Links CDN (copiar para <head>)

Bootstrap

CSS para estilos e classes utilitárias:

<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/css/bootstrap.min.css" rel="stylesheet">

JS opcional para componentes interativos (collapse, modal, dropdown...):

<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/js/bootstrap.bundle.min.js"></script>

Tailwind (CDN para desenvolvimento)

Em produção recomenda-se build local para remover classes não usadas.

<script src="https://cdn.tailwindcss.com"></script>
