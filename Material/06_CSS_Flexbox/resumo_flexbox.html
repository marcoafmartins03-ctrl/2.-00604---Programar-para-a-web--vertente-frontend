Flexbox — Resumo Visual

1. Ativar Flexbox

.container { display: flex; gap: 10px; /* espaço entre itens */ }

2. Flex Direction

Valor Direção
row Horizontal → (padrão)
row-reverse Horizontal ←
column Vertical ↓
column-reverse Vertical ↑

3. justify-content (Eixo Principal)

Valor Efeito
flex-start Início (padrão)
center Centro
flex-end Fim
space-between Espaço entre (extremos colados)
space-around Espaço à volta
space-evenly Espaço igual

4. align-items (Eixo Cruzado)

Valor Efeito
stretch Estica (padrão)
flex-start Topo
center Centro vertical
flex-end Fundo

5. Flex nos Itens

Código Efeito
flex: 1 Cresce igual aos outros
flex: 2 Cresce o dobro
flex: 0 0 200px Tamanho fixo 200px
flex: 1 1 200px Mínimo 200px, flexível

6. Código Rápido

/* Centrar perfeitamente */ .centrar { display: flex; justify-content: center; align-items: center; } /* Header com logo e menu */ .header { display: flex; justify-content: space-between; align-items: center; } /* Grid responsivo de cards */ .cards { display: flex; flex-wrap: wrap; gap: 20px; } .card { flex: 1 1 250px; } /* Sidebar fixa + conteúdo flexível */ .sidebar { flex: 0 0 200px; } .conteudo { flex: 1; }
