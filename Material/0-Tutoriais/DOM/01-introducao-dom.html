📚 Introdução ao DOM

O que é o DOM?

O Document Object Model (DOM) é uma representação em árvore da estrutura HTML de uma página web. Permite-nos manipular o conteúdo, estrutura e estilo de uma página web usando JavaScript.

// Estrutura básica do DOM document // Objeto raiz ├── html │ ├── head │ │ ├── title │ │ └── meta │ └── body │ ├── div │ ├── p │ └── button

Criar e Remover Elementos

Elemento original

// Criação de elementos const novoElemento = document.createElement('div'); novoElemento.textContent = "Novo elemento"; container.appendChild(novoElemento); // Remoção de elementos elemento.remove(); // Método moderno container.removeChild(elemento); // Método tradicional

Manipular Atributos

// Manipulação de atributos elemento.setAttribute('src', 'nova-imagem.jpg'); elemento.getAttribute('alt'); elemento.removeAttribute('title'); elemento.classList.add('nova-classe');
