CSS Layout — Resumo Visual

1. Display

Valor Comportamento Width/Height?
block Nova linha, 100% largura Sim
inline Mesma linha, só conteúdo Não
inline-block Mesma linha Sim
none Escondido -

2. Position

Valor Comportamento Referência
static Fluxo normal (padrão) -
relative Move da posição original Posição original
absolute Sai do fluxo Pai com position
fixed Fixo na janela Viewport
Dica: Para absolute funcionar, o pai precisa de position: relative!

3. Centrar Elementos

Técnica Direção Código
margin: auto Horizontal width: 300px; margin: 0 auto;
text-align Horizontal text-align: center; (no pai)
line-height Vertical height: 80px; line-height: 80px;

4. Código Rápido

/* Centrar horizontalmente */ .centrar { width: 300px; margin: 0 auto; } /* Badge no canto */ .card { position: relative; } .badge { position: absolute; top: 10px; right: 10px; } /* Botão fixo */ .fixo { position: fixed; bottom: 20px; right: 20px; }
